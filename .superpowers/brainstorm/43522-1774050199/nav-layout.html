Admin Dashboard — Sidebar Style

Pick the nav layout that feels right for the admin view

A — Icon-only slim sidebar

Compact 52px sidebar with icons only. Clean, gives maximum space to content. Hover expands to show labels. Works well on smaller screens.

B — Full labeled sidebar

160px sidebar with icon + label always visible. Classic SaaS admin feel. Easy to scan at a glance. Matches the client dashboard style.

C — Top navigation bar

Horizontal nav across the top. Maximizes vertical space. More modern, app-like feel. Better for fewer nav items (<6).
